# ربات فروش VPN (Hiddify + پرداخت کارت به کارت)

این یک ربات تلگرام حرفه‌ای برای فروش اکانت‌های پنل هیدیفای (Hiddify Manager) است که بر اساس تایید دستی ادمین (پرداخت کارت به کارت) طراحی شده است.

## ویژگی‌های کلیدی

### برای کاربران عادی
- **خرید سرویس**: انتخاب پلن، انتخاب سرور (با نمایش وضعیت فعال/غیرفعال)، و ارسال برای تایید ادمین
- **سرویس‌های من**: مشاهده لیست سرویس‌های فعال و غیرفعال، تاریخ فعال‌سازی، حجم مصرفی/باقی‌مانده، روزهای باقی‌مانده و لینک اشتراک
- **اطلاعات اکانت**: نمایش اطلاعات حساب کاربری شامل Chat ID، نام، نام کاربری، تاریخ عضویت، تعداد کل سفارشات و سرویس‌های فعال
- **راهنمای اتصال**: آموزش گام‌به‌گام برای سیستم‌عامل‌های Android، iOS و Windows با نرم‌افزار Hiddify Next و V2Box
- **چت با پشتیبانی**: ارسال پیام به ادمین و دریافت پاسخ مستقیم

### برای ادمین
- **تایید/لغو سفارش**: تایید یا لغو سفارشات کاربران
- **تمام سفارشات**: نمایش لیست تمام سفارشات با صفحه‌بندی و اطلاعات کامل
- **جستجوی سرویس**: جستجو بر اساس شماره سفارش، UUID یا لینک ساب
- **فعال‌سازی/غیرفعال‌سازی/حذف سرویس**: مدیریت سرویس‌ها
- **تمدید سرویس**: تمدید سرویس‌های کاربران (به صورت دستی برای ادمین)
- **ثبت سفارش برای کاربر**: ثبت سفارش مستقیم توسط ادمین برای کاربران (با جستجوی کاربر)
- **Inline Query**: جستجوی سریع کاربران با استفاده از اینلاین (از داخل هر چت)
- **چت دوطرفه**: دریافت پیام‌های کاربران و ارسال پاسخ به آنها
- **پیام همگانی (Broadcast)**: ارسال پیام اطلاع‌رسانی به تمام کاربران
- **بک‌آپ خودکار**: ارسال خودکار فایل دیتابیس (با بازه زمانی قابل تنظیم) به ادمین
- **بازیابی دیتابیس (Restore)**: بازیابی فایل دیتابیس از داخل ربات
- **آپدیت لیست کاربران**: اضافه کردن کاربران جدید به دیتابیس با استفاده از فایل JSON

### ویژگی‌های عمومی
- **چند سرور و لوکیشن**: پشتیبانی از چندین سرور هیدیفای با نمایش وضعیت فعال/غیرفعال
- **تقویم شمسی**: نمایش تمامی تاریخ‌ها و زمان‌ها به صورت جلالی (خورشیدی) با تایم‌زون تهران
- **آیکون‌های زیبا**: استفاده از آیکون در دکمه‌ها و پیام‌ها برای بهبود تجربه کاربری
- **امنیت آمار**: تولید شماره سفارش‌های ۶ رقمی رندوم
- **Rate Limit**: محدودیت زمانی ۲ ثانیه‌ای بین پیام‌های کاربر برای جلوگیری از اسپم
- **موازی‌سازی درخواست‌ها**: استفاده از درخواست‌های موازی برای افزایش سرعت

## پیش‌نیازها

- پنل [Hiddify Manager](https://hiddify.com) با API فعال
- توکن ربات از [@BotFather](https://t.me/BotFather)
- فعال‌سازی Inline Mode در @BotFather (/setinline)
- سرور یا سیستمی که Docker روی آن نصب باشد.

## تنظیمات فایل `.env`

فایل `.env` را بر اساس نمونه زیر ایجاد کنید:

```bash
TELEGRAM_BOT_TOKEN=your_bot_token
ADMIN_CHAT_ID=3991553456
ADMIN_USERNAME=@your_support_username
BOT_NAME=YourVPNBotName
WEBHOOK_URL=https://your-domain.com # (اختیاری - برای استفاده از وب‌هوک)
BACKUP_INTERVAL_HOURS=6
```

## پیکربندی پلن‌ها و سرورها

### ۱. پلن‌ها (`plans.json`)
لیست پلن‌های خود را در این فایل تعریف کنید (قیمت به ریال):
```json
[
  {
    "id": "10gb",
    "title": "۱۰ گیگ",
    "price_rial": 150000
  },
  {
    "id": "20gb",
    "title": "۲۰ گیگ",
    "price_rial": 250000
  },
  {
    "id": "30gb",
    "title": "۳۰ گیگ",
    "price_rial": 350000
  }
]
```

### ۲. سرورها (`servers.json`)
اطلاعات سرورهای هیدیفای خود را در این فایل وارد کنید:
```json
[
  {
    "id": "germany1",
    "title": "🇩🇪 آلمان",
    "base_url": "https://panel.example.com",
    "admin_path": "your_admin_path",
    "api_key": "your_api_key",
    "user_path": "your_user_path"
  },
  {
    "id": "usa1",
    "title": "🇺🇸 آمریکا",
    "base_url": "https://panel2.example.com",
    "admin_path": "your_admin_path",
    "api_key": "your_api_key",
    "user_path": "your_user_path"
  }
]
```

## قابلیت‌های مخصوص ادمین (با هشتگ)

- **#broadcast**: ارسال پیام همگانی (ارسال پیام و در انتهای آن #broadcast را بگذارید)
- **#search**: جستجوی سرویس (ارسال شماره سفارش/UUID/لینک ساب و در انتهای آن #search را بگذارید)
- **#restore**: بازیابی دیتابیس (آپلود فایل .db و در کپشن آن #restore را بنویسید)
- **#update_users**: آپدیت لیست کاربران (آپلود فایل JSON شامل لیست Chat ID ها و در کپشن آن #update_users را بنویسید)

## اجرا

برای اجرای پروژه با استفاده از داکر:

```bash
docker compose up -d --build
```

## ساختار پروژه

- `bot/main.py`: نقطه ورود برنامه و مدیریت سرور/وب‌هوک.
- `bot/telegram_bot.py`: منطق اصلی ربات تلگرام و هندلرها.
- `bot/db.py`: مدیریت دیتابیس SQLite (سفارشات، کاربران، لینک پیام‌ها).
- `bot/hiddify.py`: ارتباط با API پنل هیدیفای.
- `bot/config.py`: مدیریت تنظیمات و فایل‌های JSON.
